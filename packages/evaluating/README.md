# CRISP-DM – Evaluation

Conjunto de utilidades para la etapa de **Evaluation**:

- Carga del modelo entrenado y datos de validación.
- Métricas tradicionales (RMSE, MAE) y métricas por grupo (R, C).
- Reglas de decisión para criterios de negocio y sesgos.

Instala localmente con:

```bash
pip install -e packages/evaluating
```
